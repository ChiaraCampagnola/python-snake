# snakegame
Simple project (a basic game of snake) to understand how python packages work.

### Install
```pip install git+https://github.com/ChiaraCampagnola/python-snake#egg=snakegame```

### Play
```snakegame```

### Show high score
```snakegame -s```
or
```snakegame --score```

### Reset high score
```snakegame -r```
or
```snakegame --reset```
